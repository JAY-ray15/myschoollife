# 一个学期

一个极简的大学生活选择游戏。

## 玩法

- 左右二选一，做出你的选择
- 维持四个维度的平衡：精力、学业、关系、自我
- 任意维度归零，游戏结束
- 你看不到具体数值，只能通过体验感知

## 部署到静态平台

### GitHub Pages

1. 创建一个新仓库
2. 上传所有文件（`index.html`, `style.css`, `events.js`, `game.js`）
3. 在仓库设置中启用 GitHub Pages
4. 选择 main/master 分支
5. 几分钟后访问生成的链接

### Vercel

1. 登录 [vercel.com](https://vercel.com)
2. 点击 "New Project"
3. 导入这个仓库（或拖拽文件夹）
4. 点击 "Deploy"
5. 几秒钟后获得可访问的链接

### Netlify

1. 登录 [netlify.com](https://netlify.com)
2. 拖拽整个文件夹到部署区域
3. 几秒钟后获得可访问的链接

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无依赖、无后端
- 响应式设计，支持移动端

## 设计理念

少即是多。不解释，不说教。让玩家在做出选择后才意识到代价。
